# Running the webdriver functional tests

## Install NodeJS, mocha and webdriverjs

    $ brew install nodejs
    $ cd nodes/ui/test
    $ npm install mocha webdriverjs

## Install and launch selenium standalone server

Get selenium-server-standalone-* from here http://code.google.com/p/selenium/downloads/list

    $ java -jar selenium-server-standalone-2.25.0.jar
